## 🎯 各客户端详细配置教程

### **Clash for Windows** 用户：
1. 打开 Clash
2. 点击 `Profiles`（配置）
3. 粘贴任意 `.yaml` 文件的链接
4. 点击下载图标更新

### **V2RayN** 用户：
1. 打开软件
2. 点击 `订阅` → `订阅设置`
3. 添加新订阅，粘贴 `base64.txt` 链接
4. 点击 `更新订阅`

### **Shadowrocket** 用户：
1. 打开 App
2. 点击右上角 `+`
3. 选择 `Subscribe`（订阅）
4. 粘贴 `base64.txt` 链接
5. 保存

### **Quantumult X** 用户：
1. 进入 `设置` → `订阅`
2. 点击 `+`
3. 粘贴 `base64.txt` 链接
4. 备注名称，保存
